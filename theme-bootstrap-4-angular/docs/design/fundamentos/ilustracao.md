As ilustrações abaixo devem ser usadas, principalmente, para os estados de [informações vazias](https://docs.plataforma.betha.cloud/docs/design/feedback/emptyStates).

## Criando uma ilustração [​](https://docs.plataforma.betha.cloud/docs/design/fundamentos/ilustracao/\#criando-uma-ilustra%C3%A7%C3%A3o "Direct link to Criando uma ilustração")

A composição da linguagem visual é a chave para se transmitir a mensagem desejada. É necessário que os elementos utilizados sejam coerentes, criem uma unidade visual harmônica, sejam legíveis e mantenham um padrão de cores.

### Paleta [​](https://docs.plataforma.betha.cloud/docs/design/fundamentos/ilustracao/\#paleta "Direct link to Paleta")

A paleta de cores foi definida com o intuito de elaborar imagens claras, objetivas, transmitindo uma sensação de leveza, que mantenham uma coerência visual com os produtos da suíte, bem como com a identidade visual da Betha Sistemas.

![Paleta de cores](https://docs.plataforma.betha.cloud/img/ilustracao-paleta.svg)

### Grid [​](https://docs.plataforma.betha.cloud/docs/design/fundamentos/ilustracao/\#grid "Direct link to Grid")

O sistema de grid utilizado ajuda na construção das ilustrações, bem como delimita o tamanho da imagem criada.

![Grid](https://docs.plataforma.betha.cloud/img/ilustracao-grid.png)

### Traços [​](https://docs.plataforma.betha.cloud/docs/design/fundamentos/ilustracao/\#tra%C3%A7os "Direct link to Traços")

Os traços das ilustrações têm o arremate e as junções arredondadas. Deste modo, elas possuem um aspecto mais suave. Dentro do limite do grid, utilizam-se traços de 0,75pt até 8pt. As linhas mais finas são utilizadas em detalhes menores, as medianas (de 3pt a 5pt) são utilizadas na composição dos elementos principais da ilustração, já as linhas mais espessas são aplicadas nos itens que cuja ênfase é necessária.

![Traços](https://docs.plataforma.betha.cloud/img/ilustracao-tracos.svg)![Arremate e junção](https://docs.plataforma.betha.cloud/img/ilustracao-juntas.svg)

### Especificações [​](https://docs.plataforma.betha.cloud/docs/design/fundamentos/ilustracao/\#especifica%C3%A7%C3%B5es "Direct link to Especificações")

O tamanho máximo das ilustrações deve ser 250px de largura, e o tamanho mínimo 85px de largura. A escala deve ser automática, ou seja, altura proporcional à largura.
