**Open Sans** é um tipo de fonte _sans-serif_ humanista com aberturas largas em muitas letras e uma grande altura de x (letras minúsculas altas). É um tipo de letra altamente legível em telas digitais, mesmo em tamanhos pequenos. É possível utilizá-la em interfaces para dispositivos móveis e para a Web.
