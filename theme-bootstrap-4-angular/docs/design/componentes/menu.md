On this page

Este componente tem por objetivo apresentar ao usuário atalhos para as principais funcionalidades do sistema, bem como as opções de ajuda e suporte, além de outros recursos da suíte.

## Como utilizar [​](https://docs.plataforma.betha.cloud/docs/design/componentes/menu/\#como-utilizar "Direct link to Como utilizar")

O componente menu pode ser utilizado no formato vertical ou horizontal, conforme detalhado a seguir.

### Menu horizontal [​](https://docs.plataforma.betha.cloud/docs/design/componentes/menu/\#menu-horizontal "Direct link to Menu horizontal")

Esse formato é indicado aos produtos com categorias bem definidas e poucas opções de menu. Se caracteriza pela disposição de seus elementos em uma barra horizontal no topo da interface.

Quando utilizada isoladamente, a barra horizontal é composta pelos seguintes itens (da esquerda para a direita):

- Logo e nome do produto;
- Itens do menu;
- Ferramentas/funcionalidades;
- Perfil do usuário.
- A utilização dos itens que fazem parte do menu dependem da necessidade do produto, podendo tê-los ou não, além de serem acrescidos outros.

Para os itens padrões, é importante que a ordem dos itens seja mantida, para que não haja mudança brusca de um produto para o outro, auxiliando o usuário na localização dos elementos.

Observação: no nome do produto existe a opção de trocas de sistemas.

### Menu vertical [​](https://docs.plataforma.betha.cloud/docs/design/componentes/menu/\#menu-vertical "Direct link to Menu vertical")

Esse formato se caracteriza pela disposição dos itens de menu em uma barra lateral à esquerda da interface. É indicado para produtos que tenham um perfil administrativo, pois possibilita uma melhor organização dos itens e das opções de acesso aos ambientes do sistema. O componente também permite o usuário recolher ou desafixar a barra lateral da tela, aumentando o espaço de trabalho disponível.

O menu vertical é utilizado em conjunto com a barra do menu horizontal, acrescendo o ícone  à frente do nome do produto, dando acesso aos itens do menu vertical e permitindo recolhê-lo e expandi-lo.

A barra vertical é composta por:

- Itens e subitens do menu;
- Opção de fixar e desafixar o menu.
