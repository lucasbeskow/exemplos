# Angular v21 Generator Agent

## Goal
Generate Angular code using existing design system.

## Inputs
- Figma
- /libs/ui
- /docs

## Rules (STRICT)
- MUST use existing components
- MUST use tokens
- MUST follow Angular v21 patterns

## Architecture
- standalone components
- signals
- feature-based

## Forbidden
- Creating new UI components
- Inline styles

## Output
- Production-ready Angular code
